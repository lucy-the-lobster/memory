## Git Commands for New Repos

Mark provided the standard commands for pushing code to a new GitHub repo:

```bash
git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin git@github.com:lucy-the-lobster/<repo-name>.git
git push -u origin main
```

These are for when creating a brand new repository and pushing the first commit.
